## separator

**Default:** `-`

Add custom elements like svg's or icons after the `<a>` tag.

**Usage:**

```astro
---
import { Breadcrumbs } from "astro-breadcrumbs";
---

<Breadcrumbs>

  <svg
    slot="separator"
    xmlns="http://www.w3.org/2000/svg"
    width="24"
    height="24"
    viewBox="0 0 24 24"
    fill="none"
    stroke="currentColor"
    stroke-width="2"
    stroke-linecap="round"
    stroke-linejoin="round"
    ><polyline points="9 18 15 12 9 6"></polyline>
  </svg>

</Breadcrumbs>
```

**Rendered Component:**

<Breadcrumbs>

    stroke-linecap="round"

    ><polyline points="9 18 15 12 9 6"></polyline>
  </svg>

</Breadcrumbs>

## index

**Default:** `-`

[The first element] If present `indexText` is deactivated.

**Usage:**

```astro
---
import { Breadcrumbs } from "astro-breadcrumbs";
---

<Breadcrumbs>

  <svg
    slot="index"
    aria-label="Home Page"
    xmlns="http://www.w3.org/2000/svg"
    width="24"
    height="24"
    viewBox="0 0 24 24"
    fill="none"
    stroke="currentColor"
    stroke-width="2"
    stroke-linecap="round"
    stroke-linejoin="round"
    ><path d="m3 9 9-7 9 7v11a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2z">
    </path><polyline points="9 22 9 12 15 12 15 22"></polyline>
  </svg>

</Breadcrumbs>
```
